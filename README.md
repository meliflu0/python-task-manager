# python-task-manager
A simple Python CLI task manager that lets users add, view, complete, and delete tasks. Tasks are stored in a persistent .txt file so they remain saved even after closing the program. Includes an interactive menu for easy navigation.

##Features

-Interactive menu with numbered options (1, 2, 3, ...) to choose actions.
-Yes/No confirmations (y/n) where needed.
-Tasks are stored in a text file (tasks.txt) to keep data persistent.

##Main functions:

-Add tasks
-View tasks
-Complete tasks
-Delete tasks (one at a time or all at once)

##How to Use

-Clone the repository or download the files.
-Run the main program: menu.py
-Navigate the menu by entering the numbers of the options.
-For Yes/No prompts, type y for yes or n for no.
-Tasks are automatically saved in storage_file/tasks.txt.
